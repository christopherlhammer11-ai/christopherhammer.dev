# christopherhammer.dev

Personal developer portfolio for Christopher L. Hammer.

This site tells the portfolio story as one connected system: a local-first AI
workbench where HammerLock is the command center, Craig is the builder, and the
supporting repos provide memory, verification, reliability, context, precision,
and cost control.

## Local Preview

```bash
npm install
npm run dev
```

## Deploy

This is a static site. It can deploy to Vercel, GitHub Pages, Netlify, or any
static host.
