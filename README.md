# christopherhammer.dev

Personal AI engineering portfolio for Christopher L. Hammer.

This site is the front door for the GitHub portfolio: live demos, narrated proof clips, project use cases, and the story behind the local-first AI agent systems.

Live site: [https://christopherhammer.dev](https://christopherhammer.dev)

## What It Shows

- A punchy hiring headline for applied AI and agent engineering roles
- Proof-first project demos with narrated MP4s
- Use cases for HammerLock, Craig, RecallMax, Tool Use Guardian, Repo Intelligence, and related tools
- A portfolio narrative that separates flagship products, agent infrastructure, private context tools, and business applications
- Direct links to GitHub repos and live product surfaces

## Why It Exists

Recruiters and founders rarely run every repository. This site gives them the fast version:

1. What was built
2. Who it helps
3. Why it matters
4. Where to see proof

## Local Preview

This is a static site. You can open `index.html` directly or serve the folder with any static server.
